## 📋 使用说明

### 快速开始
```bash
# 一键快速选股
python src/tail_main.py --mode quick

# 或双击：尾盘选股.bat
```

### 使用推送功能
1. 访问 https://sct.ftqq.com/ 注册获取 send_key
2. 编辑 `config.py` 填入 send_key
3. 运行：`python src/tail_main.py --mode pick --push`
4. 或双击：`尾盘选股 - 带推送.bat`

### 测试数据源
```bash
python src/tail_main.py --mode test_source
# 或双击：数据源检查.bat
```
