## 解决方案

### 方案 1: 使用腾讯 API（已实现）

腾讯 API 是最稳定的数据源，已集成到以下模块：
- `src/theme_picker.py` - 题材选股器
- `src/data_sources/tencent_themes.py` - 腾讯数据源

**使用方法**:
```bash
# 查看板块热度
python src/theme_picker.py rank

# 查看特定板块选股
python src/theme_picker.py pick AI 算力
```

### 方案 2: 修改 DNS 服务器

虽然财联社 DNS 无法解析，但修改 DNS 可能改善其他网络的连接：

**Windows 手动修改**:
1. 打开"网络连接"
2. 右键当前网络 → 属性
3. 选择"Internet 协议版本 4 (TCP/IPv4)" → 属性
4. 选择"使用下面的 DNS 服务器地址"
5. 首选 DNS: `114.114.114.114`
6. 备用 DNS: `8.8.8.8`

**或使用脚本** (需要管理员权限):
```cmd
# 以管理员身份运行
D:\Projects\A\change_dns.bat
```

**恢复自动获取 DNS**:
```cmd
netsh interface ip set dns name="以太网" source=dhcp
```
