尾盘选股器 - 快速开始指南
======================================

## 安装依赖

```bash
pip install -r requirements.txt
```

## 功能特性

### 1. 多种选股策略

- **尾盘选股**：基于涨幅、量比、换手率、资金流的综合评分策略
- **强势股回调**：前期强势上涨 + 近期回调 + 站上均线的反弹机会
- **突破策略**：突破平台新高 + 放量 + 强势上涨的突破机会
- **龙虎榜策略**：基于龙虎榜数据的机构席位和游资动向选股

### 2. Server 酱微信推送

支持将选股结果推送到微信，实时掌握选股信息。

### 3. 多数据源备份

自动在 AkShare、腾讯财经、新浪财经之间切换，确保数据稳定获取。

### 4. 回测功能

支持历史回测，验证策略有效性。

## 使用方法

### 快速选股（一键运行）

```bash
python src/tail_main.py --mode quick
```

或双击 `尾盘选股.bat`（需创建）

### 运行尾盘选股策略

```bash
python src/tail_main.py --mode pick
```

### 运行其他策略

```bash
# 强势股回调
python src/tail_main.py --mode pullback

# 突破策略
python src/tail_main.py --mode breakthrough

# 龙虎榜策略
python src/tail_main.py --mode lhb

# 运行所有策略
python src/tail_main.py --mode all
```

### 开启微信推送

首次使用需先配置 Server 酱：

1. 访问 https://sct.ftqq.com/ 注册账号
2. 获取 send_key
3. 编辑 `config.py`，填入 send_key:
   ```python
   SERVERCHAN_CONFIG = {
       'send_key': 'SCTxxxxxxxxxxxxxxxxxxxx',  # 你的 send_key
       'enabled': True,
   }
   ```

运行带推送的选股：

```bash
python src/tail_main.py --mode pick --push
```

### 回测

```bash
python src/tail_main.py --mode backtest --start-date 20250101 --end-date 20251231 --hold-days 1
```

### 测试数据源

```bash
python src/tail_main.py --mode test_source
```

### 清除缓存

```bash
python src/tail_main.py --mode clear_cache
```

## 配置说明

### config.py 配置项

```python
# Server 酱推送配置
SERVERCHAN_CONFIG = {
    'send_key': '',  # 从 https://sct.ftqq.com/ 获取
    'enabled': False,
    'top_n': 10,
}

# 尾盘选股策略配置
TAIL_PICKER_CONFIG = {
    'top_n': 20,           # 返回前 N 只
    'min_gain': 3.0,       # 最小涨幅%
    'max_gain': 7.0,       # 最大涨幅%
    'min_volume_ratio': 1.5,  # 最小额比
    'min_turnover': 5.0,      # 最小换手率%
    'max_turnover': 20.0,     # 最大换手率%
    'min_main_inflow': 500,   # 最小主力净流入 (万元)
    'exclude_st': True,       # 排除 ST
}

# 强势股回调策略配置
PULLBACK_CONFIG = {
    'top_n': 20,
    'prev_gain_days': 5,      # 前期天数
    'min_prev_gain': 20.0,    # 最小额涨幅%
    'min_pullback': 5.0,      # 最小回调%
    'max_pullback': 15.0,     # 最大回调%
    'above_ma5': True,
}

# 突破策略配置
BREAKTHROUGH_CONFIG = {
    'top_n': 20,
    'breakthrough_days': 20,  # 突破 N 日新高
    'min_daily_gain': 5.0,    # 最小当日涨幅%
    'min_volume_ratio': 2.0,  # 最小量比
}

# 龙虎榜策略配置
LHB_CONFIG = {
    'top_n': 20,
    'lookback_days': 3,       # 回溯天数
    'min_institution_net_buy': 0,  # 最小机构净买入
}
```

## 输出说明

### 命令行输出

选股结果会以表格形式显示在命令行中，包含：
- 代码、名称
- 最新价、涨跌幅
- 量比、换手率
- 主力净流入
- 综合评分

### 文件输出

选股结果自动保存到 `stock_data/tail_pick/` 目录：
- CSV 格式：`pick_YYYYMMDD_HHMMSS.csv`
- Excel 格式（可选）：`pick_YYYYMMDD_HHMMSS.xlsx`

## 策略说明

### 尾盘选股策略（一夜持股法）

**核心逻辑**：选择尾盘强势、有资金推动的股票，次日冲高获利。

**筛选条件**：
- 涨幅 3%-7%（避免追高）
- 量比 >1.5（放量）
- 换手率 5%-20%（活跃但不过热）
- 主力净流入 >500 万
- 综合评分排序

### 强势股回调策略

**核心逻辑**：前期强势股回调到支撑位后的反弹机会。

**筛选条件**：
- 5 日内涨幅超过 20%
- 近期回调 5%-15%
- 当前站上 5 日线
- 5 日线趋势向上
- 量能萎缩后放大

### 突破策略

**核心逻辑**：突破平台新高的股票往往有较强上涨动能。

**筛选条件**：
- 突破 20 日/60 日新高
- 成交量放大 2 倍以上
- 当日涨幅>5%
- 平台整理至少 5 日

### 龙虎榜策略

**核心逻辑**：跟随机构席位和知名游资的动向。

**筛选条件**：
- 近 3 日登上龙虎榜
- 机构席位净买入
- 游资席位活跃

## 常见问题

### Q: Server 酱推送失败？
A: 检查 send_key 是否正确配置，确认 enabled=True。

### Q: 数据获取失败？
A: 运行 `python src/tail_main.py --mode test_source` 检查数据源状态。

### Q: 选股结果为空？
A: 可能是市场条件不符合策略要求，可适当放宽筛选条件。

### Q: 如何自定义策略参数？
A: 修改 `config.py` 中的对应配置项，或使用 `--strategy-config` 参数指定 JSON 配置文件。
