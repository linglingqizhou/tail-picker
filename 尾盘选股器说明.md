# 尾盘选股器 - 功能说明

## 已实现的功能

### 1. 核心选股策略 (`src/tail_picker.py`)

**筛选条件：**
| 条件 | 默认值 | 说明 |
|------|--------|------|
| 涨幅范围 | 3% - 7% | 避免追高和弱势股 |
| 量比 | > 1.5 | 有资金进场 |
| 换手率 | 5% - 20% | 活跃度适中 |
| 主力净流入 | > 500 万 | 主力资金流入 |
| 流通市值 | < 500 亿 | 弹性大 |
| 排除 ST | 是 | 规避风险股 |

**综合评分系统：**
- 涨幅评分 (20 分) - 越接近 5% 分越高
- 量比评分 (20 分) - 越大分越高
- 换手率评分 (15 分) - 5%-15% 最佳
- 资金流入评分 (25 分) - 流入越多分越高
- 板块热度 (20 分) - 热门板块优先

### 2. 命令行交互 (`src/tail_cli.py`)

```bash
# 快速选股
python src/tail_cli.py --mode select

# 交互式配置
python src/tail_cli.py --mode config

# 查看个股详情
python src/tail_cli.py --mode detail

# 运行回测
python src/tail_cli.py --mode backtest
```

### 3. 历史回测 (`src/backtest.py`)

**回测指标：**
- 胜率
- 总收益率
- 平均收益
- 最大回撤
- 夏普比率

```bash
# 回测最近一个月
python src/backtest.py
```

### 4. 数据导出 (`src/export.py`)

**支持格式：**
- CSV - 用于 Excel 分析
- Excel - 带格式的电子表格
- JSON - 用于程序对接
- 推送消息 - 微信/邮件格式

### 5. 主程序 (`src/tail_main.py`)

```bash
# 一键选股
python src/tail_main.py --mode quick

# 自定义选股
python src/tail_main.py --mode pick

# 回测
python src/tail_main.py --mode backtest --start-date 20250101 --end-date 20251231
```

### 6. 演示模式 (`src/tail_demo.py`)

无需联网，使用模拟数据测试功能：

```bash
python src/tail_demo.py
```
